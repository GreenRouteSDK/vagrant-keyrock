# Vagrant para keywork

## Init and start

```bash
vagrant up
```

## Stop

```bash
vagrant halt
```

## Suspend

```bash
vagrant suspend
```

## Resume

```bash
vagrant resume
```

## Access VM

```bash
vagrant ssh
```

# Inside VM

## View IDM logs

```bash
docker logs -f idm
```

## Enter docker (via bash)

```bash
docker exec -i -t idm /bin/bash
```
